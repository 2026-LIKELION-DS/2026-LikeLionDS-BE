### 📋 Commit Message Convention
| Tag | Description |
| :---: | --- |
| ✨ `Feature` | 새로운 기능 추가 |
| 🐛 `Fix` | 버그 수정 |
| 📝 `Docs` | 문서 추가, 수정, 삭제 |
| ✅ `Test` | 테스트 코드 추가, 수정, 삭제 |
| 🎨 `Style` | 마크업 및 스타일 변경 | 
| ♻️ `Refactor` | 코드 리팩토링 |
| ⚙️ `Setting` | 개발 환경 설정 관련 변경 |
| 🚀 `Deploy` | 배포 관련 변경사항 |
| 💻 `CrossBrowsing` | 브라우저 호환성 관련 변경 |
